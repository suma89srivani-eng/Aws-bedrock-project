# AWS Bedrock PDF Question Answering Project

This project is a **PDF Question Answering application** built using **Python** and **AWS Bedrock**.  
It allows users to extract content from a PDF and ask questions based on the document.

---

## Features

- Extract text from PDF files
- Ask natural language questions
- Get AI-generated answers using AWS Bedrock
- Simple Python-based implementation
